# AI-Powered Gmail Automation with n8n

## Workflow

Schedule Trigger
↓
Google Sheets
↓
Gemini AI Agent
↓
Gmail
↓
Google Sheets Update

## Overview

This workflow automatically:

- Reads data from Google Sheets
- Generates emails using Gemini AI
- Sends emails via Gmail
- Updates delivery status

## Tech Stack

- n8n
- Google Gemini
- Gmail
- Google Sheets
